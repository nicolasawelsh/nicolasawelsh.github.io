## Tools

[NetworkMiner](https://www.netresec.com/?page=NetworkMiner) - Network forensics tool to carve artifacts

[FTK Imager](https://www.exterro.com/digital-forensics-software/ftk-imager) - Image mounter

[Arsenal Image Mounter](https://arsenalrecon.com/products/arsenal-image-mounter) - Image mounter

[KAPE](https://www.kroll.com/en/insights/publications/cyber/kroll-artifact-parser-extractor-kape) - Triage platform

[ZimmermanTools](https://ericzimmerman.github.io/#!index.md) - DFIR tool suite

[Hayabusa](https://github.com/Yamato-Security/hayabusa) - Sigma-based evtx hunting

[Volatility3](https://github.com/volatilityfoundation/volatility3) - Memory analysis

[MemProcFS](https://github.com/ufrisk/MemProcFS) - Automates a lot of tediousness of memory analysis
- Used in conjunction with [MemProcFS-Analyzer](https://github.com/evild3ad/MemProcFS-Analyzer)

[Autopsy](https://www.autopsy.com/) - Image analysis
