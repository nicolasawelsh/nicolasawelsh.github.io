# DFIR-Kit-Deployment

This guide set walks through building a practical DFIR lab foundation. In digital forensics and incident response work, you often want an isolated environment where you can safely ingest, store, and analyze evidence.

The NAS and ELK stack deployments are the first steps in preparing that environment. This example deployment is built around an ESXi node.
